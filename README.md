# 📊 Student Performance Analysis using Excel

## 📌 Project Overview

This project focuses on analyzing student performance using a dataset titled **StudentPerformanceFactors**. The goal is to identify key factors that influence student exam scores and derive actionable insights using Microsoft Excel.

This case study demonstrates how data analysis techniques can be applied in the education domain to improve academic outcomes.

---

## 🎯 Problem Statement

Educational institutions often struggle to understand which factors significantly impact student performance.

This project aims to answer:

> **What are the key factors affecting student exam scores, and how can data-driven insights help improve student performance?**

---

## 🎯 Objectives

* Identify factors influencing **Exam_Score**
* Analyze impact of:

  * Study habits (Hours_Studied, Sleep_Hours)
  * Attendance
  * Motivation_Level
  * Tutoring_Sessions
* Compare **high vs low performers**
* Provide actionable insights for improving results

## 🧹 Data Cleaning & Preparation (Excel)

* Removed duplicate records
* Handled missing values using:

  * Mean (for numerical columns)
  * Mode (for categorical columns)
* Converted categorical variables into structured groups:

  * Motivation_Level (Low, Medium, High)
  * Sleep Groups (<6, 6–8, >8 hours)
* Created helper columns for better analysis

## 💡 Key Insights

* Students with higher **study hours** tend to score better
* **Attendance above 80%** significantly improves performance
* **Optimal sleep (6–8 hours)** is associated with higher scores
* **Motivation level** strongly impacts exam results
* Students attending **tutoring sessions** perform better on average

---

## 🚀 Recommendations

* Encourage consistent study schedules
* Promote healthy sleep habits (6–8 hours)
* Increase access to tutoring support
* Improve attendance tracking systems
* Support students with low motivation through mentorship programs
